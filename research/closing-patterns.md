# 第3問 決算問題 頻出論点

## 概要

- **配点**: 35点（試験全体の約35%）
- **制限時間の目安**: 25〜30分（試験全体60分中）
- **目標点数**: 24〜28点（約70〜80%）
- **問題の性質**: 決算整理前残高試算表＋決算整理事項を与えられ、決算整理仕訳を反映させる応用問題
- **戦略**: 決算整理仕訳を反射的にできる状態にすることが最優先。各論点のパターンを暗記し、素早く正確に処理する。

---

## 出題形式の3パターン

### 1. 財務諸表（貸借対照表・損益計算書）の作成

- **出題頻度**: 最多。過去12回中6回（約50%）
- **特徴**: 決算整理後の数値をB/S・P/Lの形式で記入する。当期純利益がB/SとP/Lで一致するか検算できる。
- **難所**: B/SとP/Lで科目の表示区分（流動資産、固定資産、流動負債など）を正しく選ぶ必要がある。

### 2. 精算表の作成

- **出題頻度**: 2番目。過去12回中4回（約33%）
- **特徴**: 修正記入欄（決算整理仕訳欄）と損益計算書欄・貸借対照表欄に転記する多段式の表。記入欄が多く、ミスが目立ちやすい。
- **難所**: 「文章形式」と「推定形式」の2種類がある。推定形式はパズル的に逆算が必要。

### 3. 決算整理後残高試算表の作成

- **出題頻度**: 最少。過去12回中2回（約17%）。ただしネット試験での出題報告あり。
- **特徴**: 決算整理仕訳を適用した後の各勘定残高を一覧表示する。精算表より構造はシンプル。
- **難所**: 経過勘定など「新しく生まれる勘定科目」の金額を1から計算する必要がある。

---

## 決算整理仕訳 頻出論点 Top10

---

### 1. 売上原価の算定

- **頻出度**: ★★★（ほぼ毎回出題）
- **概要**: 期中は商品を「繰越商品（資産）」と「仕入（費用）」で管理しており、期末に売上原価を正しく計算するため、期首・期末の在庫を仕入勘定へ振り替える。
- **典型的な問題文例**:
  > 「期末商品の棚卸高は¥50,000である。なお、売上原価は仕入勘定で計算すること。」
- **仕訳**（「しーくり、くりしー」で暗記）:
  ```
  （借）仕入    30,000  （貸）繰越商品  30,000  ← 期首商品を仕入へ（し→くり）
  （借）繰越商品 50,000  （貸）仕入     50,000  ← 期末商品を繰越商品へ（くり→し）
  ```
- **計算式**: 売上原価 = 期首商品棚卸高 + 当期仕入高 − 期末商品棚卸高

---

### 2. 貸倒引当金の設定（差額補充法）

- **頻出度**: ★★★（ほぼ毎回出題）
- **概要**: 売掛金や受取手形などの売上債権のうち、将来回収できない可能性がある金額を見積もり、費用として計上する。既存の引当金残高との「差額」だけを追加設定するのが差額補充法。
- **典型的な問題文例**:
  > 「売掛金の期末残高¥200,000に対し、2%の貸倒引当金を差額補充法で設定する。なお、貸倒引当金の残高は¥1,000である。」
- **仕訳**:
  ```
  必要額：200,000 × 2% = 4,000
  追加額：4,000 − 1,000（既存残高）= 3,000

  （借）貸倒引当金繰入  3,000  （貸）貸倒引当金  3,000
  ```
- **計算式**: 繰入額 = （売上債権残高 × 設定率）− 貸倒引当金既存残高

---

### 3. 減価償却（定額法・間接法）

- **頻出度**: ★★★（ほぼ毎回出題）
- **概要**: 固定資産の価値が年々減少する分を費用として計上する。簿記3級では定額法（毎年同額）・間接法（累計額を別勘定で管理）が標準。
- **典型的な問題文例**:
  > 「備品（取得原価¥300,000、残存価額ゼロ、耐用年数5年）について定額法で減価償却を行う。」
- **仕訳**:
  ```
  年間償却額：（300,000 − 0）÷ 5 = 60,000

  （借）減価償却費  60,000  （貸）備品減価償却累計額  60,000
  ```
- **計算式**: 年間償却額 = （取得原価 − 残存価額）÷ 耐用年数
- **注意**: 残存価額が「取得原価の10%」と指定されるケースもある。月割り計算（期中取得の場合）を求められることもある。

---

### 4. 経過勘定（前払費用・前受収益・未払費用・未収収益）

- **頻出度**: ★★★（ほぼ毎回出題。4種類のうち2〜3種類が同時出題されることが多い）
- **概要**: 費用・収益を「発生した期」に正しく帰属させるための調整。

| 種類 | 意味 | 分類 |
|------|------|------|
| 前払費用 | 翌期分をすでに払っている | 資産 |
| 未払費用 | 当期分がまだ支払われていない | 負債 |
| 前受収益 | 翌期分をすでに受け取っている | 負債 |
| 未収収益 | 当期分がまだ受け取られていない | 資産 |

- **典型的な問題文例（前払費用）**:
  > 「保険料のうち¥12,000は、翌期4月から9月分（6か月分）である。」
- **仕訳（前払費用）**:
  ```
  （借）前払保険料  12,000  （貸）保険料  12,000
  ```
- **典型的な問題文例（未払費用）**:
  > 「給料のうち当月分¥30,000が翌月払いのため未払いとなっている。」
- **仕訳（未払費用）**:
  ```
  （借）給料  30,000  （貸）未払給料  30,000
  ```
- **典型的な問題文例（前受収益）**:
  > 「受取家賃¥60,000のうち翌期分が¥20,000含まれている。」
- **仕訳（前受収益）**:
  ```
  （借）受取家賃  20,000  （貸）前受家賃  20,000
  ```
- **典型的な問題文例（未収収益）**:
  > 「貸付金¥100,000に対する利息（年利2%）のうち、当期分（6か月）が未収となっている。」
- **仕訳（未収収益）**:
  ```
  利息：100,000 × 2% × 6/12 = 1,000

  （借）未収利息  1,000  （貸）受取利息  1,000
  ```

---

### 5. 現金過不足の整理

- **頻出度**: ★★（比較的よく出題）
- **概要**: 期中に現金の帳簿残高と実際残高に差が生じた場合、期末決算時に原因不明の差額を「雑損」または「雑益」として処理する。
- **典型的な問題文例**:
  > 「現金過不足¥500（借方残高）の原因が不明であったため、雑損として処理する。」
- **仕訳（帳簿>実際→現金が足りない→雑損）**:
  ```
  （借）雑損  500  （貸）現金過不足  500
  ```
- **仕訳（実際>帳簿→現金が多い→雑益）**:
  ```
  （借）現金過不足  500  （貸）雑益  500
  ```
- **注意**: 期中仕訳（現金過不足の発見）と決算整理仕訳（雑損/雑益への振替）を混同しないこと。

---

### 6. 当座借越の振替

- **頻出度**: ★★（出題あり）
- **概要**: 当座預金が貸方残高（マイナス）になっている場合、決算日に負債勘定「当座借越」または「借入金」へ振り替える。翌期首に再振替仕訳で戻す。
- **典型的な問題文例**:
  > 「当座預金勘定が貸方残高¥50,000となっている。当座借越として処理すること。」
- **仕訳（決算整理）**:
  ```
  （借）当座預金  50,000  （貸）当座借越  50,000
  ```
- **翌期首の再振替仕訳**:
  ```
  （借）当座借越  50,000  （貸）当座預金  50,000
  ```
- **注意**: 問題によっては「借入金」勘定を使うこともある。問題文の指示に従う。

---

### 7. 消耗品・貯蔵品の処理

- **頻出度**: ★★（出題あり）
- **概要**: 期末に残っている未使用の消耗品を資産（消耗品または貯蔵品）へ振り替える。または期首に資産計上されていた消耗品を当期使用分として費用へ振り替える。

**パターンA：消耗品費で購入→未使用分を資産へ振替**
- **典型的な問題文例**:
  > 「消耗品費¥10,000のうち、期末時点で¥3,000が未使用である。」
- **仕訳**:
  ```
  （借）消耗品  3,000  （貸）消耗品費  3,000
  ```

**パターンB：消耗品（資産）で購入→使用分を費用へ振替**
- **典型的な問題文例**:
  > 「期首に消耗品¥10,000を購入（資産計上）。期末の残高は¥3,000である。」
- **仕訳**:
  ```
  使用額：10,000 − 3,000 = 7,000
  （借）消耗品費  7,000  （貸）消耗品  7,000
  ```

---

### 8. 法人税等の計上

- **頻出度**: ★★（出題あり）
- **概要**: 当期の確定した利益に対する法人税・住民税・事業税を費用として計上し、未払分を負債に計上する。中間納付分がある場合は差し引く。
- **典型的な問題文例**:
  > 「当期の法人税等は¥120,000と計算された。なお、中間納付額¥50,000は仮払法人税等として処理済みである。」
- **仕訳**:
  ```
  （借）法人税、住民税及び事業税  120,000
      （貸）仮払法人税等           50,000
      （貸）未払法人税等           70,000
  ```
- **中間納付がない場合**:
  ```
  （借）法人税、住民税及び事業税  120,000  （貸）未払法人税等  120,000
  ```

---

### 9. 消費税の処理（税抜方式）

- **頻出度**: ★★（出題あり）
- **概要**: 期中に計上した「仮払消費税（資産）」と「仮受消費税（負債）」を相殺し、差額を「未払消費税（負債）」として確定する。
- **典型的な問題文例**:
  > 「消費税を確定すること。仮払消費税¥600,000、仮受消費税¥1,000,000。」
- **仕訳**:
  ```
  納付額：1,000,000 − 600,000 = 400,000

  （借）仮受消費税  1,000,000
      （貸）仮払消費税    600,000
      （貸）未払消費税    400,000
  ```
- **注意**: 仮払消費税が仮受消費税を上回る場合は「未収消費税（資産）」になる（稀）。

---

### 10. 損益振替・繰越利益剰余金への振替（決算振替仕訳）

- **頻出度**: ★★（財務諸表作成形式で理解が必要。精算表では直接入力しないケースもある）
- **概要**: 期末に収益・費用勘定を損益勘定へ振り替えて当期純利益を確定し、その金額を繰越利益剰余金勘定へ振り替える。これにより収益・費用勘定はゼロリセットされ、次期に引き継がれる。
- **仕訳ステップ1（収益の振替）**:
  ```
  （借）売上   xxx  （貸）損益   xxx
  （借）受取利息 xx  （貸）損益   xx
  ```
- **仕訳ステップ2（費用の振替）**:
  ```
  （借）損益  xxx  （貸）仕入     xxx
  （借）損益  xx   （貸）給料     xx
  （借）損益  xx   （貸）減価償却費 xx
  ```
- **仕訳ステップ3（当期純利益を繰越利益剰余金へ）**:
  ```
  当期純利益の場合：
  （借）損益  300,000  （貸）繰越利益剰余金  300,000

  当期純損失の場合：
  （借）繰越利益剰余金  200,000  （貸）損益  200,000
  ```

---

## 解答プロセス（典型）

第3問は「出てきた順番に処理する」が基本原則。

1. **問題全体を30秒で俯瞰する**
   - 出題形式（精算表 / 財務諸表 / 決算整理後残高試算表）を確認
   - 決算整理事項の個数と論点を把握
   - 配点を意識し、時間内に解ける量かを判断

2. **決算整理前残高試算表の数値を把握する**
   - 各勘定の残高をざっと確認
   - 経過勘定や消費税など、後で調整が入る勘定に印をつける

3. **決算整理事項を1つずつ順番に処理する**（典型的な出題順）
   - 現金過不足の処理
   - 当座借越の振替
   - 売上原価の算定（しーくり・くりしー）
   - 貸倒引当金の設定
   - 減価償却の計上
   - 経過勘定（前払・未払・前受・未収）の処理
   - 消耗品・貯蔵品の処理
   - 消費税の確定
   - 法人税等の計上

4. **集計・転記**
   - 決算整理後の各残高を計算し、解答欄へ記入
   - 精算表の場合：修正記入欄 → 損益計算書欄 → 貸借対照表欄 の順に転記

5. **検算**
   - 財務諸表の場合：B/SとP/Lの当期純利益が一致するか確認
   - 精算表の場合：損益計算書欄・貸借対照表欄の借方合計＝貸方合計を確認

## 部分点戦略

### 時間配分の目安

| 作業 | 目安時間 |
|------|---------|
| 問題の俯瞰・方針決定 | 1〜2分 |
| 決算整理仕訳の処理 | 15〜18分 |
| 集計・転記 | 5〜8分 |
| 検算 | 2〜3分 |
| **合計** | **25〜30分** |

### 部分点を最大化するコツ

1. **簡単な論点を確実に取る**: 現金過不足・消耗品・当座借越などシンプルな論点は落とさない。売上原価・貸倒引当金・減価償却は必ず満点を狙う。

2. **計算根拠を下書きに残す**: メモ欄や余白に計算式を書いておく。数値を間違えても計算式が正しければ部分点が入ることがある（試験形式による）。

3. **わからない論点はいったん飛ばす**: 時間を使いすぎず、後でわかる論点を確実に取る。

4. **B/Sの当期純利益で誤りを発見する**: P/LとB/Sの当期純利益が一致しない場合、差額が「誤りの金額の2倍」になる場合は借方・貸方の入れ違いが疑われる。

5. **勘定科目名のミスに注意**: 「貸倒引当金繰入」と「貸倒引当金」、「減価償却費」と「備品減価償却累計額」など、ペアで使う科目を正確に書く。

6. **精算表の推定形式は後回し**: 推定形式（逆算パズル）は時間がかかるため、確実に取れる論点を先に処理する。
