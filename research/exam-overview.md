# 日商簿記3級 試験概要

## 基本情報

| 項目 | 内容 |
|------|------|
| 試験時間 | 60分 |
| 満点 | 100点 |
| 合格基準 | 70点以上 |
| 受験料（統一試験） | 3,300円（税込） |
| 受験料（ネット試験） | 3,300円（税込）＋事務手数料 550円（税込） |
| 対象科目 | 商業簿記 |

※受験料は2024年4月1日改定（物価高対応）

---

## 試験形式の違い

| 項目 | 統一試験（紙） | ネット試験（CBT） |
|------|--------------|----------------|
| 実施頻度 | 年3回（6月・11月・2月） | 随時（ほぼ毎日） |
| 申込方法 | 各地商工会議所 | CBT-Solutions ポータル |
| 結果通知 | 後日（約1ヶ月） | 試験終了直後に画面表示 |
| 出題形式 | 記述式中心 | 選択式＋入力式（3題以内） |
| 出題範囲 | 同一 | 同一 |
| 難易度傾向 | やや難化回あり | 基本問題中心でやや安定 |

ネット試験には「統一試験前後の施行休止期間」がある点に注意。

---

## 出題構成

| 大問 | 配点 | 内容 |
|------|------|------|
| 第1問 | 45点 | 仕訳問題 15問 × 3点（3級全範囲から出題） |
| 第2問 | 20点 | 補助簿の選択・勘定記入・用語選択など |
| 第3問 | 35点 | 精算表・試算表・損益計算書・貸借対照表などの総合問題 |

- 合計：100点満点
- 70点以上で合格

## 出題範囲（概要）

現行範囲（2019年改定以降）は**株式会社を前提とした商業簿記**。主なトピック：

- 仕訳・勘定記入（現金・預金・売掛金・買掛金・手形・電子記録債権債務など）
- クレジット売掛金、差入保証金
- 株式会社の設立・剰余金の配当
- 法人税等・消費税（税抜方式）
- 補助簿（売上帳・仕入帳・売掛金元帳・固定資産台帳など）
- 試算表・精算表・財務諸表の作成
- 決算整理（減価償却・引当金・繰延・見越など）

詳細論点は別途調査予定。

---

## 最新の改定動向

### 2019年改定（主要改定）

旧制度との主な変更点：

| 変更 | 内容 |
|------|------|
| 想定対象 | 個人商店 → **株式会社** |
| 問題構成 | 5問構成 → **3問構成** |
| 追加項目 | 株式発行・剰余金配当、電子記録債権債務、クレジット売掛金、法人税等、消費税（税抜）、固定資産台帳、決算整理後残高試算表 |
| 廃止項目 | 有価証券の取得・売却、受取配当金、減価償却の直接法、引出金、繰越試算表 |

### 2024年改定

- 受験料を 2,200円（税込）→ **3,300円（税込）** に値上げ（2024年4月1日施行）
- 出題範囲の変更なし

### 2025年改定

- 出題範囲の変更なし（2024年度と同一）

### 2027年改定（予告済み）

2027年4月1日施行予定。主な変更点：

| 変更 | 内容 |
|------|------|
| 手形の廃止 | 紙の手形・小切手廃止に伴い手形・債権譲渡枠を削除 |
| 普通預金の明記 | 「現金預金」項目に普通預金を明記 |
| キャッシュレス対応 | クレジット売掛金に「キャッシュレス決済を含む」追記 |
| 2級→3級へ移行 | 売上原価勘定への振替、固定資産の除却・定率法（基礎レベルのみ） |
