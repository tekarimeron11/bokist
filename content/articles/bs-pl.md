# 貸借対照表・損益計算書の作り方

> 第3問の出題形式で **最頻出** が「B/S・P/L 作成」。精算表より一見ややこしく見えますが、やっていることは精算表とほぼ同じ。今日は B/S と P/L の作成手順を1記事で完全解説します。

## A. ストーリー

第3問の出題形式は3パターン（[決算整理仕訳の全体像](/articles/closing-entry-flow) 参照）。そのうち約50%を占めるのが **財務諸表（B/S と P/L）作成形式** です。

問題を見ると、決算整理 **前** 残高試算表と決算整理事項が与えられ、解答用紙には B/S と P/L の枠だけが置かれている。先輩はそこに **決算整理後の残高を、所定の表示区分で書き入れる** わけです。

精算表との違いは2つだけ：

1. 解答欄が **2つの完成形式（B/S・P/L）** になっている
2. **表示区分（流動・固定など）** や **名称（売掛金 → 受取手形 etc）** が問われる

それ以外の作業は同じ。今日はこの形式特有のコツを押さえに行きます。

## B. 用語の整理

- **貸借対照表（B/S）**: 期末時点で会社が持っている資産・負債・純資産のスナップショット。読みは「Balance Sheet」。
- **損益計算書（P/L）**: その期間に稼いだ収益・使った費用・差額の利益を並べた表。「Profit and Loss Statement」。
- **流動資産（りゅうどうしさん）／固定資産（こていしさん）**: 1年以内にお金になる資産が流動、1年以上使う資産が固定。表示区分の違い。
- **流動負債／固定負債**: 1年以内に返す負債が流動、1年超が固定。
- **当期純利益**: 収益 − 費用。P/L の最後に出てきて、B/S の繰越利益剰余金にも加算される。
- **表示用の名称**: 簿記上の勘定名と B/S・P/L 上の表示名は **微妙に違う** ことがあります（後述）。

### 解く手順（5ステップ）

1. **決算整理事項を順に処理** して、各勘定の決算整理後残高を計算する（精算表と同じ）
2. 各勘定を **B/S と P/L のどちらに書くか** を判断（5分類で振り分け）
3. **表示区分**（流動・固定）と **表示名** の変換を確認
4. **当期純利益** を P/L の差額として求め、B/S の繰越利益剰余金にも加算
5. **B/S 借方合計 = 貸方合計** を検算

### 表示区分のルール

簿記3級では、B/S は **流動資産 → 固定資産 → 流動負債 → 固定負債 → 純資産** の順に表示する形式が出ます（「報告式」）。1年基準でざっくりこんな振り分け。

| 区分 | 主な勘定 |
|------|---------|
| 流動資産 | 現金・普通預金・受取手形・売掛金・繰越商品・前払費用・未収収益 |
| 固定資産 | 建物・備品・土地・車両運搬具（減価償却累計額を控除して表示） |
| 流動負債 | 買掛金・支払手形・未払金・未払費用・前受収益・未払消費税・未払法人税等 |
| 固定負債 | 長期借入金（3級ではほぼ出ない） |
| 純資産 | 資本金・繰越利益剰余金 |

3級では **「1年以内に返す or 受け取る」かどうか** で流動・固定を判断する場面はあまりなく、勘定名で機械的に振り分けることが多いです。

### 表示名の変換に注意

簿記上の勘定名と、B/S・P/L 上の表示名が **違う** ことがあります。3級でよく出るのは：

| 簿記上の勘定 | B/S・P/L 上の表示 |
|------------|----------------|
| 仕入 | **売上原価** |
| 繰越商品 | **商品** |
| 貸倒引当金（売掛金分） | **売掛金から控除する形で表示** |
| 減価償却累計額 | **建物 / 備品から控除する形で表示** |
| 受取家賃・受取利息 | **そのまま** |

「仕入勘定の残高」を P/L に書くときは、**売上原価** という表示名で書く決まりです。試験で勘定名のまま書いてしまうと減点されます。

### 評価勘定の表示

貸倒引当金や減価償却累計額は **資産から控除する形** で B/S に表示します。

```
（売掛金まわり）
売掛金             200,000
貸倒引当金         △ 4,000   ← マイナス記号で控除
─────────────────
売掛金（純額）     196,000

（備品まわり）
備品               300,000
減価償却累計額     △120,000   ← マイナス記号で控除
─────────────────
備品（純額）       180,000
```

「△」記号や「( )」で囲んで「控除する」と表すのが慣例。試験の解答用紙には **既に控除欄が用意されている** ことが多いので、そこに金額を書き入れるだけです。

### 当期純利益で B/S と P/L をつなぐ

P/L で計算した **当期純利益** は、B/S の **繰越利益剰余金に加算** します。

```
P/L:    収益合計 805,000 − 費用合計 650,000 = 当期純利益 155,000

B/S:    繰越利益剰余金（試算表残）100,000
        + 当期純利益                155,000
        ─────────────────────────────
        繰越利益剰余金（B/S）       255,000
```

P/L の最終行に「当期純利益 155,000」、B/S の繰越利益剰余金欄に「255,000」を書き込めば完成です。

### 検算：当期純利益が B/S と P/L で一致するか

財務諸表作成形式の最大のメリットは、**B/S と P/L で当期純利益が一致するか** で全体の検算ができること。

```
B/S 借方合計 − B/S 貸方合計（当期純利益除く）= 当期純利益（B/S側で算出）
P/L 貸方合計 − P/L 借方合計 = 当期純利益（P/L側で算出）

両者が一致 → 正解の可能性大
不一致 → どこかにミスあり
```

不一致の場合は、まず **「差額が誤りの2倍か」** をチェック。借方・貸方の入れ違いだと差額が誤りの2倍になることが多いです。

### 精算表との違い

精算表（[精算表の埋め方](/articles/worksheet-flow) 参照）との違いを整理しておきます。

| 項目 | 精算表 | B/S・P/L 作成 |
|------|------|-----------|
| 解答欄の形 | 8桁の表 | B/S と P/L の完成形式 |
| 修正記入欄 | あり（決算整理仕訳を明示） | なし（脳内で処理） |
| 表示名の変換 | 不要（勘定名そのまま） | 必要（仕入→売上原価 など） |
| 表示区分（流動・固定） | 不要 | 必要 |
| 当期純利益の書き方 | P/L 借方＋B/S 貸方 | P/L 末尾＋繰越利益剰余金へ加算 |

精算表は機械的に埋めればOKですが、B/S・P/L 作成形式は **表示の知識が必要** です。これが「ややこしい」と感じる原因になっています。

### よく出る論点と表示

第3問でよく出る論点ごとに、B/S・P/L での書き方をまとめておきます。

| 決算整理 | B/S 表示 | P/L 表示 |
|---------|--------|--------|
| 売上原価の算定 | 商品（繰越商品の期末残） | 売上原価 |
| 貸倒引当金の設定 | 売掛金から△控除 | 貸倒引当金繰入（販管費） |
| 減価償却 | 累計額から△控除 | 減価償却費（販管費） |
| 経過勘定（前払・未払・前受・未収） | 流動資産または流動負債 | 該当する費用・収益で調整済み |
| 消耗品・貯蔵品 | 流動資産（消耗品・貯蔵品） | 消耗品費・通信費で調整済み |
| 消費税 | 未払消費税（流動負債） | P/L には現れない |
| 法人税等 | 未払法人税等（流動負債） | 法人税、住民税及び事業税 |
| 当期純利益 | 繰越利益剰余金に加算 | P/L 末尾に表示 |

## D. ポイント整理

- 解く手順は **決算整理 → 振り分け → 表示変換 → 当期純利益 → 検算**
- B/S は **流動資産 → 固定資産 → 流動負債 → 固定負債 → 純資産** の順
- 表示名の変換に注意：**仕入 → 売上原価**、**繰越商品 → 商品**
- 評価勘定（貸倒引当金・減価償却累計額）は **△控除** で表示
- **当期純利益** は B/S と P/L の両方で一致する（検算ポイント）
- 不一致のときは **差額が誤りの2倍** かを確認すると借貸入れ違いが見つけやすい
- 精算表より **表示の知識** が必要
