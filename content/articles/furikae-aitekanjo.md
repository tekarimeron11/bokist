# 振替と相手勘定

> 仕訳の左右は必ずペアで動きます。そのペア関係を呼ぶ言葉が「振替」と「相手勘定」。意味を押さえると、訂正仕訳もすんなり理解できます。

## A. ストーリー

仕訳には必ず **借方（左）と貸方（右）の両方** に勘定科目が並びます。たとえば「現金80,000円で商品を仕入れた」ならこう。

```
借方: 仕入 80,000
貸方: 現金 80,000
```

このとき「仕入」から見ると **相手勘定は現金**、「現金」から見ると **相手勘定は仕入** です。仕訳の左右は **常に何かと何かのペア** で動いている、というのが今日のお話。

そして、ある勘定から別の勘定へ **金額をスライドさせる** ような仕訳のことを **振替（ふりかえ）** といいます。決算整理や訂正仕訳でよく出てくる重要な概念です。

## B. 用語の整理

- **仕訳（しわけ）**: 取引を借方と貸方に分けて記録する作業。詳しくは [複式簿記とはなにか](/articles/double-entry) で。
- **借方／貸方**: 仕訳の **左側／右側** の欄。詳しくは [借方・貸方の本当の意味](/articles/debit-credit) で。
- **相手勘定（あいてかんじょう）**: 仕訳の中で、ある勘定とペアになっているもう一方の勘定。
- **振替（ふりかえ）**: ある勘定から別の勘定へ、金額を **付け替える** 仕訳。
- **逆仕訳（ぎゃくしわけ）**: もとの仕訳の借方と貸方を **入れ替えた** 仕訳。打ち消す働きをします。
- **訂正仕訳（ていせいしわけ）**: 過去にした仕訳の誤りを直すための仕訳。

## C. 本論

### 「相手勘定」の見え方

仕訳の左右は必ず何かとペアです。それぞれの勘定から見て、もう片方を「相手勘定」と呼びます。

たとえば **「商品100,000円をクレジットカードで販売、手数料3%」** の仕訳：

```
借方: クレジット売掛金 97,000
      支払手数料        3,000
貸方: 売上            100,000
```

- **売上**（収益）から見ると、相手勘定は **クレジット売掛金**（資産）と **支払手数料**（費用）の2つ
- **クレジット売掛金**から見ると、相手勘定は **売上**

複数行ある場合、相手勘定が複数いる、と捉えます。総勘定元帳（あとで [仕訳から財務諸表まで](/articles/from-journal-to-statements) で扱います）では、相手勘定が複数のときに **「諸口（しょくち）」** と書く習慣もあります。

> **諸口（しょくち）**: 「相手勘定がたくさんある」ことをひとことで表す摘要欄の表現。

### 「振替」とは

**振替** はもう少し動きのある言葉で、**ある勘定から別の勘定へ金額を付け替える** 仕訳のこと。お金そのものは動かなくても、帳簿上のラベルだけ貼り替える、というイメージです。

代表的な振替の例：

**(1) 当座預金から小口現金への前渡し**

```
借方: 小口現金 30,000
貸方: 当座預金 30,000
```

会社のお金が当座預金から小口現金に **位置だけ変わった** 取引。資産から資産への振替です。

**(2) 売掛金から電子記録債権への振替**

```
借方: 電子記録債権 80,000
貸方: 売掛金       80,000
```

「あとでもらえる権利」のラベルを売掛金から電子記録債権に貼り替えただけ。これも資産同士の振替。

**(3) 決算整理の経過勘定**

```
借方: 前払保険料 14,000
貸方: 保険料     14,000
```

「保険料（費用）として書いていた金額のうち14,000円は来期の分」という付け替え。費用から資産への振替です。

決算整理の多くは **何らかの振替** で構成されています。

### 逆仕訳：もとの仕訳を打ち消す

振替の手前にある重要概念が **逆仕訳**。**もとの仕訳の借方と貸方を入れ替えた** 仕訳のことで、もとの仕訳を **「なかったこと」にする** 働きがあります。

たとえば「商品15,000円を掛けで売った」の仕訳：

```
（もとの仕訳）
借方: 売掛金 15,000
貸方: 売上   15,000
```

これを打ち消す逆仕訳はこう。

```
（逆仕訳）
借方: 売上   15,000
貸方: 売掛金 15,000
```

借方と貸方を **入れ替えただけ**。両方を合算すると借方も貸方も「売掛金 +0、売上 +0」となり、もとの取引が完全に取り消されます。

返品の仕訳が **逆仕訳と同じ形** になるのはそのためです。「売った話をなかったことにする」のが返品なので。

### 訂正仕訳：誤りを直す技

過去の仕訳に誤りがあったとき、それを直すのが **訂正仕訳**。基本の考え方は **「逆仕訳で誤りを打ち消す」＋「正しい仕訳を立てる」** の2段構え。実務では1本にまとめて書きます。

**(例)** 消耗品5,000円を現金で買ったのに、誤って「消耗品費500/現金500」と書いていた。

```
（誤った仕訳）
借方: 消耗品費 500
貸方: 現金     500
```

これを打ち消す逆仕訳と、正しい仕訳を並べると：

```
（逆仕訳：誤りを取り消す）
借方: 現金     500
貸方: 消耗品費 500

（正しい仕訳）
借方: 消耗品費 5,000
貸方: 現金     5,000
```

これを足し合わせると、現金は「+500−5,000＝−4,500」、消耗品費は「−500+5,000＝+4,500」。1本にまとめると：

```
（訂正仕訳）
借方: 消耗品費 4,500
貸方: 現金     4,500
```

つまり訂正仕訳は **「正しい状態 − 今の状態」の差分だけを書く** 仕訳になります。金額誤りでも勘定科目誤りでも、考え方は同じです。

> 試験では金額誤り・勘定科目誤りの両方が出ます。手元で「逆仕訳＋正仕訳」を書き出してから合算するのが確実です。

## D. ポイント整理

- 仕訳の左右は **必ずペア**。相手側を **相手勘定** と呼ぶ
- **振替** は、ある勘定から別の勘定へ金額を付け替える仕訳。決算整理の多くがこの形
- **逆仕訳** はもとの仕訳の借方／貸方を入れ替えたもの。打ち消す働き
- **訂正仕訳** は「逆仕訳＋正しい仕訳」を1本にまとめた **差分仕訳**
- 返品の仕訳が逆仕訳の形になるのは「なかったことにする」から
