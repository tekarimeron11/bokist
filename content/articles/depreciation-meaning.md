# 減価償却の本当の意味

> 「減価償却ってよく聞くけど、何のためにやってるの？」と聞かれてパッと答えられたら、半分わかっています。今日はそこに踏み込みます。

## A. ストーリー

先輩がカフェを開くために、エスプレッソマシン（備品）を300,000円で買ったとします。これは1日で消えるものではなく、**5年くらい使い続けて少しずつ古くなっていく** 物です。

ここで疑問。買った日に「300,000円の費用がドカン」と全部記録するのが正しいでしょうか？それとも「毎年60,000円ずつ、価値が減っていく分を費用に」と分割して記録するべき？

簿記の答えは **後者** です。これが **減価償却（げんかしょうきゃく）** という考え方。なぜ分割するのか、どう計算するのか、なぜ「累計額」という変な勘定を使うのか。順に見ていきます。

## B. 用語の整理

- **固定資産（こていしさん）**: 1年以上使う物のこと。建物・備品・車両・土地など。
- **取得原価（しゅとくげんか）**: 固定資産を手に入れるのにかかった総額。本体価格＋付随費用（運賃・設置費・仲介手数料など）。
- **耐用年数（たいようねんすう）**: 「だいたい何年使える物か」という見積もりの年数。簿記の問題では問題文に書いてあります。
- **残存価額（ざんぞんかがく）**: 耐用年数を使い切ったあとに残ると見積もる価値。3級では基本ゼロ。
- **減価償却（げんかしょうきゃく）**: 固定資産の取得原価を、使用期間にわたって **少しずつ費用に** していく手続き。
- **定額法（ていがくほう）**: 毎年同じ金額ずつ減るとして計算する方法。3級ではこれだけ。
- **間接法（かんせつほう）**: 減価償却の記帳方法のひとつで、固定資産そのものは減らさず、別勘定 **減価償却累計額** に減少分を積み上げていく方式。3級ではこれを使います。
- **減価償却累計額**: これまで償却してきた金額の **累計**。固定資産の **マイナス側** で表示する評価勘定です。

## C. 本論

### なぜ一括ではなく分割するのか：費用と収益の対応

減価償却の根っこにあるのは **費用と収益を、同じ期間に対応させる** という考え方。会計では **費用収益対応の原則** と呼ばれます。

エスプレッソマシンを5年使うとして、その5年のあいだに **コーヒーを売って稼ぐ収益** が立ちますよね。マシン代を「買った日に全部費用」にしてしまうと、その年だけドカンと費用が出て、収益とのバランスが取れません。

```
（一括費用にした場合）
1年目: 売上 500,000、費用 300,000（マシン代だけで）→ 利益200,000
2〜5年目: 売上 各500,000、費用 0 → 各年 利益 500,000
```

この見え方では、1年目だけ妙に利益が小さく、2年目以降は急に大きく見えてしまいます。実際の経営実態とはかけ離れていますよね。

そこで、**マシンが稼ぎ続ける5年間に、その費用も5年に分けて配分する** のが減価償却。

```
（5年に分割した場合）
1年目: 売上 500,000、費用 60,000 → 利益 440,000
2〜5年目: 売上 各500,000、費用 60,000 → 各年 利益 440,000
```

毎年同じくらいの儲けが出ているように見える。これが正しい姿です。

### 定額法の計算

3級では **定額法** という、毎年同じ金額ずつ減らす方式を使います。計算式は：

```
1年あたりの減価償却費 ＝ （取得原価 − 残存価額）÷ 耐用年数
```

例：取得原価 300,000円、残存価額 0、耐用年数 5年なら

```
（300,000 − 0）÷ 5 ＝ 60,000円
```

毎年60,000円ずつ費用にしていきます。

### 期中取得は月割で

期の途中で買った場合は **月割計算** が必要です。「3月決算で12月に買った」なら、12月から3月までの **4か月だけ** が今期の使用期間。

```
年額 60,000 × 4/12 ＝ 20,000円（今期の費用）
```

問題文に「期首取得」とあれば1年分まるまる、「○月取得」とあれば月割、と判断します。

### 仕訳：間接法という変わった書き方

減価償却の記帳には **直接法** と **間接法** という2つのやり方があります。3級では **間接法** を使うので、まず直接法と比較してから見ていきましょう。

**(直接法)**

```
借方: 減価償却費 60,000
貸方: 備品       60,000
```

備品の金額そのものを減らす、シンプルな書き方。

**(間接法)** ← 3級はこちら

```
借方: 減価償却費      60,000
貸方: 減価償却累計額  60,000
```

備品本体には触らず、**減価償却累計額** という別勘定に「減った分」を積み上げていく書き方。

なぜわざわざ間接法を使うのか？理由は2つあります。

1. **取得原価の情報を残せる**: 直接法だと年々備品の金額が減って、当初いくらで買ったかが見えなくなります。間接法なら備品は300,000円のまま、累計額が60,000、120,000、180,000……と積み上がるので、「いくらで買って、いくら償却済みか」が一目でわかります。
2. **残存価額の管理がラク**: 帳簿価額（取得原価−累計額）を計算するだけで現在の評価額が出せます。

### 売却時に累計額を消す

固定資産を売却するとき、**それまで積み上げた累計額を一気に取り崩す** 必要があります。たとえば「備品（取得原価600,000、累計額200,000）を450,000で売却、代金は月末受取」なら：

```
借方: 未収入金        450,000
      減価償却累計額  200,000
貸方: 備品            600,000
      固定資産売却益   50,000
```

備品を消し（600,000）、それまで積み上げていた累計額も消し（200,000）、受け取る代金（450,000）を未収入金で立て、差額（450,000−（600,000−200,000）＝50,000）を売却益として計上します。

### 減価償却の対象にならないもの

すべての固定資産が減価償却の対象になるわけではありません。**土地** は減価償却しません。これは土地が **使っても価値が減らない** （と簿記では考える）ためです。建物や備品が減価償却の対象、土地は対象外、と区別して覚えてください。

## D. ポイント整理

- 減価償却は **固定資産の費用を、使用期間にわたって配分する** 手続き
- 根っこの考え方は **費用収益対応の原則**（収益と費用を同じ期間に揃える）
- 3級では **定額法**（毎年同じ金額）と **間接法**（累計額勘定を使う）の組み合わせ
- 計算式: **(取得原価 − 残存価額) ÷ 耐用年数 ＝ 1年分**
- 期中取得は **月割計算**
- 売却時は **累計額も一緒に取り崩す**
- **土地は減価償却の対象外**
