# 棚卸減耗損・商品評価損 — 数量と価額のロスを分けて記録する

> 「商品の数を数えたら帳簿より少なかった」「売れる値段が下がった」。決算ではこの2種類のロスを **別の勘定** で記録します。混ざりやすいですがコツがあります。

## A. ストーリー

[売上原価の算定](/articles/cost-of-goods-sold) で「しーくり、くりしー」を使って繰越商品を確定させたとします。期末に残っている商品 ¥80,000（100個 × @¥800）が次期に持ち越されるはず。

ところが先輩が倉庫を実地で数えてみたら、**実際には95個しかなかった**。さらに、その商品は近年人気が落ちていて **市場価格が @¥780 まで下がっている**。

このとき帳簿の ¥80,000 をそのまま B/S に載せるのはウソになります。**「減った数量分」と「下がった価値分」を別々に費用計上** して、繰越商品を実態に合わせて減らす。これが今日のテーマです。

## B. 用語の整理

- **帳簿棚卸高（ちょうぼたなおろしだか）**: 帳簿上にあるはずの商品金額。「単価 × 帳簿数量」で計算します。
- **実地棚卸高（じっちたなおろしだか）**: 倉庫で実際に数えた商品金額。「単価 × 実際数量」で計算します。
- **棚卸減耗損（たなおろしげんもうそん）**: 帳簿数量より実地数量が少ないときの差額分の費用。盗難・紛失・破損などが原因。
- **商品評価損（しょうひんひょうかそん）**: 実地数量に残っている商品の **価値が下がった** 分の費用。市場価格の下落などが原因。
- **正味売却価額（しょうみばいきゃくかがく）**: 実際に売れる予想価格。これが原価より低いと評価損が出ます。

## C. 本論

### ロスは2種類ある

棚卸減耗損と商品評価損は、別の原因で起きる別のロスです。混ざらないよう **2軸で整理** します。

| 種類 | 何が起きた？ | 原因 | 計算式 |
|------|----------|------|--------|
| **棚卸減耗損** | 数量が減った | 盗難・紛失・破損 | （帳簿数量 − 実地数量）× 原価 |
| **商品評価損** | 価値が下がった | 市場価格の下落・劣化 | 実地数量 × （原価 − 正味売却価額） |

数量のズレ → 棚卸減耗損、価額のズレ → 商品評価損、と覚えてください。

### 計算する順番

商品評価損の計算には **「実地数量」** を使います。なので **必ず棚卸減耗損を先に出す** のがコツです。

```
1. 帳簿数量と実地数量の差から → 棚卸減耗損
2. 実地数量に残った商品の単価差から → 商品評価損
```

冒頭のストーリーで具体的に計算してみます。

```
帳簿: 100個 × @¥800 = ¥80,000
実地: 95個 × @¥780（正味売却価額）

【棚卸減耗損】
（100個 − 95個）× @¥800 = 5個 × @¥800 = ¥4,000

【商品評価損】
95個 × （@¥800 − @¥780）= 95個 × @¥20 = ¥1,900

合計 ¥5,900 だけ繰越商品を減らす
```

### 仕訳の書き方

両方を1つの仕訳にまとめると、こうなります。

```
借方: 棚卸減耗損   4,000
      商品評価損   1,900
貸方: 繰越商品     5,900
```

借方には2つの費用を並べ、貸方は繰越商品を合計額（¥5,900）でまとめて減らします。

棚卸減耗損だけ出る問題（評価損なし）も多いです。その場合はもっとシンプル。

```
（棚卸減耗損のみのとき）
借方: 棚卸減耗損   3,000
貸方: 繰越商品     3,000
```

商品評価損だけ出る問題はあまり見かけませんが、考え方は同じです。

### しーくり、くりしーの後にやる

順番に注意。決算整理では **先に「しーくり、くりしー」で繰越商品を期末帳簿棚卸高に確定させてから**、その上で減耗・評価損の調整をします。

```
1. しーくり、くりしーで → 繰越商品 ¥80,000（帳簿棚卸高）
2. 棚卸減耗損 ¥4,000、商品評価損 ¥1,900 で → 繰越商品 ¥74,100 に減額
3. B/S の繰越商品 → ¥74,100
```

最終的に B/S に載るのは **¥74,100**（実地数量 × 正味売却価額）になります。

### P/L 上の表示

棚卸減耗損と商品評価損は、3級では基本 **販売費及び一般管理費** や **売上原価の内訳** として表示されます（試験では「販管費区分」と「売上原価区分」のどちらに入れるかは問題指示に従う）。費用であることに変わりはないので、出題側からどう書くか指示があるはずです。

### 紛失・盗難の場合の言い回し

問題文では「棚卸減耗が判明した」「盗難により減耗が生じた」「保管中の破損により……」など、原因の書き方がいろいろあります。原因が何であれ、**「数量差」なら棚卸減耗損** の一択。原因にとらわれずに数量差を見つけることが大切です。

## D. ポイント整理

- ロスは **数量差（棚卸減耗損）** と **価額差（商品評価損）** の2種類
- 計算は **必ず棚卸減耗損から**（評価損には実地数量を使うため）
- 棚卸減耗損 = (帳簿数量 − 実地数量) × **原価**
- 商品評価損 = 実地数量 × (原価 − **正味売却価額**)
- 借方に両方の費用を並べ、貸方は繰越商品を合計で減らす
- **しーくり、くりしーの後** にやる順番
