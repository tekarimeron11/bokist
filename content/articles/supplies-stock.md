# 消耗品と貯蔵品の使い分け

> 「コピー用紙の残りはどうするの？」「郵便切手が余ってたら？」決算で必ず聞かれる消耗品・貯蔵品の振替を、ややこしい言葉も含めて1記事で整理します。

## A. ストーリー

先輩のカフェ「Bean Garden」で、1年間に **コピー用紙やレシート用紙を ¥10,000 ぶん買いました**。当然、買ったときは費用「消耗品費」として処理しています。でも、年度末に倉庫を見たら **¥3,000 ぶん未使用で残っている**。

このとき、決算では「未使用分は来期に持ち越す資産」として整理する必要があります。これが今日のテーマ。さらに、**郵便切手や収入印紙** も似た理由で別の振替をします。両者の仕分けを覚えれば、3級の決算整理1論点を確実に取れます。

## B. 用語の整理

- **消耗品費（しょうもうひんひ）**: コピー用紙・ペン・レシート用紙など、すぐ使い切る事務用品の費用勘定。
- **消耗品（しょうもうひん）**: 期末に残った未使用の事務用品を翌期に持ち越すための **資産** 勘定。
- **通信費（つうしんひ）**: 郵便切手・電話代・宅配便代などの費用勘定。
- **貯蔵品（ちょぞうひん）**: 期末に残った **郵便切手・収入印紙** などを翌期に持ち越すための **資産** 勘定。簿記3級では切手・印紙の置き場所として登場。
- **収入印紙（しゅうにゅういんし）**: 契約書や領収書に貼る印紙。買ったときは「租税公課」または「通信費」（実務では租税公課が一般的、3級では問題文の指示に従う）。

## C. 本論

### なぜ振替が必要なの？

消耗品も切手も、**買ったときに費用** にしておきます（その方が普段の処理がラクなので）。

```
（期中：購入時）
借方: 消耗品費   10,000
貸方: 現金       10,000
```

でも年度末に **未使用ぶん** が残っていれば、それは「使ってないのに費用にしている」状態。これは事実とズレているので、決算で **使った分だけが費用、未使用分は資産** に振り替えます。

> **発生主義**: 簿記の基本ルール。費用は「お金を払った日」ではなく「実際に使った日」に計上する。買ったけど使っていない部分は当期の費用ではない、という考え方です。

### 消耗品（コピー用紙・ペンなど）の振替

期中に消耗品費 ¥10,000、期末に ¥3,000 が未使用なら：

```
借方: 消耗品   3,000      ← 資産が増える
貸方: 消耗品費 3,000      ← 費用を打ち消す
```

これで当期の費用は ¥10,000 − ¥3,000 = **¥7,000**、来期に持ち越す資産は **¥3,000** になります。来期の期首には逆仕訳（再振替）を切って、資産から費用に戻します。

```
（翌期首：再振替）
借方: 消耗品費   3,000
貸方: 消耗品     3,000
```

### 貯蔵品（切手・印紙）の振替

切手や印紙は「通信費」または「租税公課」として **買ったときに費用** にしています。期末に未使用分が残っていれば、**貯蔵品** という資産勘定に振り替えます。

```
（期末：振替）
借方: 貯蔵品   4,000      ← 資産が増える
貸方: 通信費   4,000      ← 費用を打ち消す
```

期首の再振替も同じ要領。

```
（翌期首：再振替）
借方: 通信費   4,000
貸方: 貯蔵品   4,000
```

### 消耗品と貯蔵品、どっちを使う？

問題文に出てくるモノで判断します。

| もの | 振替先（資産） | 元の費用 |
|------|--------------|---------|
| コピー用紙、ペン、レシート用紙、文具一般 | **消耗品** | 消耗品費 |
| 郵便切手 | **貯蔵品** | 通信費 |
| 収入印紙 | **貯蔵品** | 租税公課（または通信費） |
| 葉書、切手台紙 | **貯蔵品** | 通信費 |

ざっくり覚えると **「事務用品系は消耗品、切手・印紙系は貯蔵品」**。なお、租税公課は3級の出題範囲では使わない問題集も多く、収入印紙も「通信費 / 貯蔵品」で出題されることがあります。問題文の指示が最優先です。

### 期首の再振替を忘れない

期末に振り替えた資産は、**翌期首にもう一度費用に戻す** のが定番ルール。これをやっておかないと、来期に切手を使ったときに二重計上になってしまいます。

期首の再振替は、期末仕訳を **そっくり逆向きに書く** だけ。問題文に「期首にあたり再振替仕訳を行う」と指示があれば、迷わず逆仕訳を書いてください。

```
（期末）借方: 貯蔵品 / 貸方: 通信費
（期首）借方: 通信費 / 貸方: 貯蔵品  ← 借方・貸方を入れ替える
```

### 別パターン：購入時に資産で記帳しているケース

実務でも稀にありますが、**買ったときに資産（消耗品・貯蔵品）として記帳していた** 場合は、決算で **使用分を費用に振り替える** 逆向きの処理になります。

```
（期中）借方: 消耗品 10,000 / 貸方: 現金 10,000
（期末）借方: 消耗品費 7,000 / 貸方: 消耗品 7,000   ← 使った分を費用に
```

3級では基本「期中は費用で買って、期末に未使用分を資産へ」のパターンがメインですが、問題文によっては逆パターンも出ます。**問題文に「未使用分」とあるか「使用分」とあるか** で判断してください。

### 試験での出題形式

第3問の決算整理事項で、こんな1行で出てきます。

> 「通信費勘定の残高には、期末時点で未使用の郵便切手 ¥4,000 が含まれている。これを貯蔵品勘定へ振り替える。」

落ち着いて「未使用」というキーワードを見つけたら、機械的に **貯蔵品 / 通信費** の振替を切ればOK。配点的には小さい論点ですが、確実に取れる **部分点ゾーン** です。

## D. ポイント整理

- 期末に残った **未使用分** は資産に振り替える（発生主義）
- 事務用品系（コピー用紙・ペン）→ **消耗品（資産）**
- 切手・印紙系 → **貯蔵品（資産）**
- 振替仕訳は「資産（借方）／元の費用（貸方）」のセット
- **期首の再振替** を忘れない（期末仕訳を逆向きに）
- 配点は小さいが落とさない **部分点ゾーン**
