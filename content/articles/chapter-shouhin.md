# 商品売買の仕訳パターン徹底解説

> 第1問15問のうち2〜3問は商品売買から。ここを落とすと致命的です。逆にいえば、押さえれば確実に得点源になります。

## 商品売買は「三分法」で処理する

3級では、商品の出入りを **「仕入」「売上」「繰越商品」** の3つの勘定で処理します。これを **三分法（さんぶんぽう）** と呼びます。

> 💡 **三分法**: 商品の動きを「仕入（費用）」「売上（収益）」「繰越商品（資産）」の3勘定で記録する方法。3級ではこれだけ覚えればOK。

## パターン1：仕入のとき

商品を買ったら **借方に「仕入」**、貸方は支払い方法によって変わります。

| 支払い方法 | 貸方 |
|----------|-----|
| 現金で払った | 現金 |
| ツケにした | 買掛金（負債） |
| 約束手形を振り出した | 支払手形（負債） |

> 💡 **買掛金（かいかけきん）**: 仕入のツケ。あとで払う約束＝負債です。

### 諸掛り（しょがかり）に注意

引取運賃など、商品の入手にかかった**付随費用は仕入の金額に含めます**。

例：「50,000円の商品を仕入れ、運賃2,000円を現金で支払い、商品代金は掛け」

```
借方: 仕入   52,000
貸方: 買掛金 50,000
      現金    2,000
```

> 💡 **諸掛り**: 商品の運賃や手数料など、買う／売るときにかかる付随費用のこと。

## パターン2：売上のとき

商品を売ったら **貸方に「売上」**、借方は受け取り方法で変わります。

| 受取方法 | 借方 |
|---------|-----|
| 現金でもらった | 現金 |
| ツケにした | 売掛金（資産） |
| 約束手形を受け取った | 受取手形（資産） |

> 💡 **売掛金（うりかけきん）**: 売上のツケ。あとで受け取る権利＝資産です。

## パターン3：返品・値引

返品は **元の仕訳を逆向きに書く** だけ。シンプルです。

例：「掛けで仕入れた商品10,000円を返品した」

```
借方: 買掛金 10,000
貸方: 仕入   10,000
```

例：「掛けで売った商品15,000円が返品された」

```
借方: 売上   15,000
貸方: 売掛金 15,000
```

## パターン4：クレジット売掛金

クレジットカード払いの売上は、**通常の売掛金と区別**して「クレジット売掛金」を使います。信販会社への手数料は **支払手数料（費用）** で計上します。

例：「100,000円をクレジットで販売、手数料3%」

```
借方: クレジット売掛金 97,000
      支払手数料        3,000
貸方: 売上            100,000
```

> 💡 **クレジット売掛金**: クレジットカード会社経由で回収する売掛金のこと。手数料を引いた額が借方に立つのがポイント。

## つまずきポイント

- 諸掛りを **仕入金額に含め忘れる**（特に引取運賃）
- クレジット売掛金で **手数料を貸方に書いてしまう**（手数料は費用だから借方）
- 返品で **元の仕訳と同じ向き**に書いてしまう（逆向きが正解）

## まとめ

- 3級の商品売買は **三分法**（仕入・売上・繰越商品）
- 仕入の **諸掛り** は仕入金額に含める
- 返品は **元の仕訳を逆向き** に書くだけ
- クレジット売掛金は **手数料を借方の支払手数料で計上**

## この記事で出てきた言葉

- **三分法**: 商品売買を3勘定で処理する方法
- **買掛金／売掛金**: 仕入／売上のツケ。負債／資産
- **諸掛り**: 引取運賃など商品取引の付随費用
- **クレジット売掛金**: クレカ決済の売上で使う専用勘定
- **支払手数料**: 信販会社などに払う手数料（費用）
