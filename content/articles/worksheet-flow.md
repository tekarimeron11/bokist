# 精算表の埋め方

> 8桁精算表は **「決算整理を整理しながら B/S と P/L を作るワークシート」**。8列の動き方さえわかれば、第3問の精算表問題は反射的に解けます。

## A. ストーリー

簿記3級の第3問で出る精算表は、**8つの列** で構成された大きな表です。最初に見ると圧倒されますが、構造はシンプル。

```
| 勘定科目 | 試算表 | 修正記入 | 損益計算書 | 貸借対照表 |
|         | 借 貸  | 借 貸    | 借 貸      | 借 貸     |
```

左から右へ、**「決算整理前 → 修正 → P/L または B/S に振り分け」** という流れで残高を運んでいきます。今日はこの8列の動きを完全マスターします。

## B. 用語の整理

- **精算表（せいさんひょう）**: 決算整理仕訳を反映しながら、試算表から B/S・P/L を作るためのワークシート。
- **8桁精算表（はちけたせいさんひょう）**: 試算表・修正記入・P/L・B/S の4ペア×借方／貸方 = 計8列の精算表。3級で使う標準形式。
- **試算表（しさんひょう）**: 決算整理 **前** の各勘定残高を借方・貸方に並べた表。精算表の出発点。
- **修正記入（しゅうせいきにゅう）**: 決算整理仕訳の借方・貸方の金額を書き込む列。決算整理事項を順に処理した結果がここに入る。
- **P/L（損益計算書）欄 / B/S（貸借対照表）欄**: 決算整理後の残高を、**収益・費用 → P/L、資産・負債・純資産 → B/S** に振り分けて記入する列。
- **当期純利益**: P/L 欄と B/S 欄の最後に金額を書き入れる、精算表の締めの一行。

## C. 本論

### 8列の役割

8列がそれぞれ何をする欄か、まずは一覧化します。

| 列 | 中身 | 役割 |
|---|------|------|
| ① 試算表 借方 | 期中残高（資産・費用がここ） | 出発点 |
| ② 試算表 貸方 | 期中残高（負債・純資産・収益がここ） | 出発点 |
| ③ 修正記入 借方 | 決算整理仕訳の借方 | 整理を反映 |
| ④ 修正記入 貸方 | 決算整理仕訳の貸方 | 整理を反映 |
| ⑤ P/L 借方 | 決算整理後の費用 | 損益計算書の借方 |
| ⑥ P/L 貸方 | 決算整理後の収益 | 損益計算書の貸方 |
| ⑦ B/S 借方 | 決算整理後の資産 | 貸借対照表の借方 |
| ⑧ B/S 貸方 | 決算整理後の負債・純資産 | 貸借対照表の貸方 |

ポイントは2つ。

- **①②** は決算整理 **前** の状態（出発点）
- **⑤⑥⑦⑧** は決算整理 **後** の状態（ゴール）

そして③④（修正記入）が両者をつなぐ役目を果たします。

1. **試算表** は問題文に既に書き込まれている（解く側は基本写すだけ）
2. **修正記入** に決算整理仕訳をすべて書き込む
3. 試算表 ± 修正記入 を計算して、勘定の種類ごとに **P/L 欄** か **B/S 欄** に振り分ける
4. P/L と B/S の借方・貸方に差額（当期純利益）を書き込んで合計を一致させる

### 修正記入の書き方

決算整理事項に出てくる仕訳を、そのまま **修正記入欄** に書き込みます。たとえば「決算整理事項：保険料のうち未経過分 ¥6,000 を前払処理する」なら、

```
決算整理仕訳:
  借方: 前払保険料  6,000
  貸方: 保険料      6,000
```

これを精算表の修正記入欄に書きます。

```
| 勘定科目  | 試算表 借 | 試算表 貸 | 修正 借 | 修正 貸 | …
| 前払保険料| —        | —        | 6,000   | —       |
| 保険料    | 24,000   | —        | —       | 6,000   |
```

決算整理事項を上から順に処理し、すべての修正記入を書き終えたら、**修正記入 借方合計 = 修正記入 貸方合計** になっているか必ず確認します。一致していなければ、どこかでミスしています。

### P/L 欄と B/S 欄への振り分け

修正記入が終わったら、各勘定の **試算表 + 修正記入** を計算して、P/L 欄か B/S 欄のどちらかに書き込みます。

| 勘定の種類 | どちらの欄へ？ |
|----------|--------------|
| 資産（現金・売掛金・繰越商品 など） | **B/S 借方** |
| 負債（買掛金・借入金 など） | **B/S 貸方** |
| 純資産（資本金・繰越利益剰余金） | **B/S 貸方** |
| 収益（売上・受取利息 など） | **P/L 貸方** |
| 費用（仕入・給料・減価償却費 など） | **P/L 借方** |

5分類のホームポジション（[5分類のホームポジション](/articles/home-position) 参照）と完全に一致しています。

### 当期純利益を書き込む

最後の仕上げ。**P/L と B/S の合計を一致させる行**として、当期純利益（または当期純損失）を書き込みます。

P/L 欄を例に説明すると：

```
P/L 借方合計（費用合計）= 650,000
P/L 貸方合計（収益合計）= 805,000
差額 = 155,000（収益 > 費用なので当期純利益）
```

差額 ¥155,000 を **P/L 借方の最終行** に「当期純利益 155,000」として書き入れて、合計を ¥805,000 で揃えます。

同時に **B/S 貸方の最終行** にも「当期純利益 155,000」を書き入れます。これで B/S の借方合計と貸方合計も一致するはず。

```
P/L:    借方 650,000 + 当期純利益 155,000 = 貸方 805,000 ✓
B/S:    借方 990,000 = 貸方 835,000 + 当期純利益 155,000 ✓
```

両方の表で同じ ¥155,000 が登場するのが正解の証拠。これは精算表問題の **検算ポイント** として超重要です。

### 当期純損失のとき

費用 > 収益で赤字の場合、書き方が逆になります。

```
P/L 貸方の最終行: 当期純損失 40,000
B/S 借方の最終行: 当期純損失 40,000
```

**「黒字なら P/L 借方／B/S 貸方」「赤字なら P/L 貸方／B/S 借方」** と覚えてください。両方とも「合計を揃える行」として機能します。

### よくあるミスと対処

精算表を解いていて合計が合わないとき、典型的な原因は3つ。

1. **修正記入欄の借方・貸方が一致していない**: 決算整理仕訳のどれかが借方or貸方の金額を間違えている。修正記入合計から検算する。
2. **修正額の符号を間違えている**: 試算表残高に対して足すべきところを引いている、またはその逆。
3. **勘定の振り分けミス**: 評価勘定（貸倒引当金・減価償却累計額）を B/S 借方に書いてしまうなど。これらは **B/S 貸方** に書きます。

差額が出たら、まず **修正記入合計の一致** から確認するのが鉄則です。

## D. ポイント整理

- 8桁精算表は **試算表 → 修正記入 → P/L・B/S** の3ペア×借貸 = 8列
- 埋める順番は **試算表 → 修正記入 → P/L・B/S → 当期純利益**
- 修正記入欄の **借方合計 = 貸方合計** を必ずチェック
- 振り分けは **5分類のホームポジション** どおり（資産→B/S借、収益→P/L貸 など）
- 当期純利益（黒字）：**P/L 借方 + B/S 貸方** に書き入れて両方の合計を揃える
- 当期純損失（赤字）：**P/L 貸方 + B/S 借方** に書き入れる
- 評価勘定（貸倒引当金・減価償却累計額）は **B/S 貸方** に書く
