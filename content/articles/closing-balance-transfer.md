# 損益振替・繰越利益剰余金への振替

> 決算整理が終わったあとに最後にやる **「決算振替」** という3行の仕訳。何のためにやるのか、なぜ「損益」勘定が登場するのか、ここで完全に納得しに行きます。

## A. ストーリー

決算整理仕訳（[決算整理仕訳の全体像](/articles/closing-entry-flow) で扱った10論点）が全部終わったとします。各勘定の残高は確定しました。でも、まだ「**当期にいくら儲かったか**」を確定させる仕訳が残っています。それが **決算振替仕訳** です。

具体的には3つのステップ：

1. 収益（売上・受取利息など）を全部 **「損益」勘定の貸方** に集める
2. 費用（仕入・給料・減価償却費など）を全部 **「損益」勘定の借方** に集める
3. 損益勘定の差額（=当期純利益）を **「繰越利益剰余金」** に移す

この3ステップで、収益と費用の勘定はすべて **ゼロにリセット** され、来期はゼロからスタートできるようになります。

## B. 用語の整理

- **決算振替仕訳（けっさんふりかえしわけ）**: 決算整理が終わった後に、収益・費用を損益勘定に集めて当期純利益を確定させる仕訳。
- **損益勘定（そんえきかんじょう）**: 決算のときだけ登場する **集計用の特殊勘定**。期中はどこにも出てこない。略して「P/L sumary」「pl_summary」とも書かれます。
- **当期純利益（とうきじゅんりえき）**: 当期に実際に儲かった金額。= 収益合計 − 費用合計。
- **繰越利益剰余金（くりこしりえきじょうよきん）**: 過去から積み上げた利益の累計を表す **純資産** の勘定。当期純利益はここに加算されます。
- **当期純損失（とうきじゅんそんしつ）**: 費用が収益より多かったときの赤字。繰越利益剰余金から減算されます。

### ステップ① 収益を損益へ

収益勘定（売上・受取利息・受取家賃など）は普段 **貸方** に積み上がっています。これを全部「損益勘定」の貸方に集めます。

仕訳の書き方は **収益勘定の残高を借方に書いて打ち消し、相手勘定として「損益」を貸方に立てる** イメージ。

```
（例：売上 800,000、受取利息 5,000）
借方: 売上       800,000
      受取利息     5,000
貸方: 損益       805,000
```

これで売上勘定・受取利息勘定はゼロになり、損益勘定の貸方に ¥805,000 が積み上がります。

### ステップ② 費用を損益へ

費用勘定（仕入・給料・減価償却費・支払家賃など）は普段 **借方** に積み上がっています。これも全部「損益勘定」へ集めます。

仕訳は **費用勘定の残高を貸方に書いて打ち消し、相手勘定として「損益」を借方に立てる**。

```
（例：仕入 500,000、給料 120,000、減価償却費 30,000）
借方: 損益       650,000
貸方: 仕入       500,000
      給料       120,000
      減価償却費   30,000
```

これで費用勘定はゼロ、損益勘定の借方には ¥650,000 が積み上がります。

### この時点での損益勘定

ステップ①②が終わった時点で、損益勘定は次のような姿です。

```
損益勘定
─────────────────────
借方: 650,000 │ 貸方: 805,000
（費用合計）  │ （収益合計）
─────────────────────
差額 155,000（貸方残） ← 当期純利益
```

貸方の方が大きい場合は **当期純利益**、借方の方が大きい場合は **当期純損失** です。

### ステップ③ 繰越利益剰余金へ

最後に、損益勘定の差額（当期純利益）を **繰越利益剰余金** に移します。

```
（当期純利益のとき）
借方: 損益              155,000
貸方: 繰越利益剰余金    155,000
```

借方に「損益」を書いて貸方残高を打ち消し、純資産の繰越利益剰余金を増やします。これで損益勘定もゼロになり、すべての勘定が次期へクリーンに引き継がれる状態になります。

### 当期純損失のときは逆向き

費用 > 収益 だった場合は赤字です。繰越利益剰余金を取り崩すので、仕訳の向きが逆になります。

```
（当期純損失 40,000 のとき）
借方: 繰越利益剰余金   40,000
貸方: 損益             40,000
```

借方の繰越利益剰余金は「純資産が減る」、貸方の損益は「借方残を打ち消す」イメージ。当期純利益のパターンを覚えておけば、損失のときは **借方・貸方を入れ替えるだけ** で対応できます。

### 損益勘定はどこに行く？

決算振替が終わると、損益勘定は **ゼロ** になります。来期からはまた空っぽの状態でスタート。決算のときだけ一時的に存在する集計箱、と覚えてください。

> **損益勘定**: 期中は登場せず、決算のときだけ収益・費用を集めるために使う特殊な勘定。集計が終わるとゼロになり次期に持ち越されない。

繰越利益剰余金は逆に、**毎期どんどん積み上がっていく純資産** です。過去の利益の累積をここに記録しておくことで、配当の原資になったり、B/S の純資産部分に「これまでの蓄積」として表示されたりします。

### 精算表ではどう表現する？

精算表の問題では、**この決算振替仕訳を明示的に書かないこと** が多いです。代わりに、

- P/L 欄: 収益と費用の差額を「当期純利益」として借方欄に書く（収益>費用のとき）
- B/S 欄: 同じ金額を「繰越利益剰余金」の貸方欄に追加する

という対応で、決算振替を表現します。財務諸表作成形式の問題でも同じで、「P/L の最終行に当期純利益、B/S の純資産に繰越利益剰余金として組み込む」だけでOK。仕訳問題（第1問）として明示的に問われるときに、上の3ステップを書く形になります。

## D. ポイント整理

- 決算振替は **3ステップ**：①収益を損益へ ②費用を損益へ ③損益を繰越利益剰余金へ
- **損益勘定** は決算だけの一時的な集計箱。期中は登場しない
- ① 収益（普段貸方残） → 借方に書いて消し、貸方に「損益」
- ② 費用（普段借方残） → 貸方に書いて消し、借方に「損益」
- ③ **当期純利益** → 借方損益／貸方繰越利益剰余金
- ③ **当期純損失** → 借方繰越利益剰余金／貸方損益（向き逆）
- 精算表問題では明示しないことも多いが、考え方は同じ
