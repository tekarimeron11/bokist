# 仕訳から財務諸表まで

> 「仕訳って書いたあとどうなるの？」と思ったことのある先輩へ。仕訳から最終的な成績表（財務諸表）までは、決まったレールの上を進みます。今日はその全体像を。

## A. ストーリー

簿記には **取引→仕訳→元帳→試算表→精算表→財務諸表** という処理の流れがあります。1日の取引を仕訳に落とし、それが集計されて、最後に **貸借対照表（B/S）と損益計算書（P/L）** という会社の成績表になります。

実務ではほぼコンピューターが自動でやってくれますが、簿記3級ではこの **流れの理屈** を理解しておくことが大切です。試験では各段階のどこかが出題されるので、全体像を頭に入れておくと、問題を見たときに「あ、これは試算表のステップだな」とすぐ位置づけられるようになります。

## B. 用語の整理

- **取引（とりひき）**: 簿記の世界での「お金や物が動いたできごと」全部。
- **仕訳（しわけ）**: 取引を借方／貸方に分けて記録する作業。詳しくは [複式簿記とはなにか](/articles/double-entry) で。
- **仕訳帳（しわけちょう）**: 仕訳を発生順に並べて書く帳簿。
- **総勘定元帳（そうかんじょうもとちょう）**: 勘定科目ごとに金額を集計しておく帳簿。短く **元帳（もとちょう）** とも呼びます。
- **試算表（しさんひょう）**: 元帳の各勘定残高を一覧にした表。集計が合っているか確認するためのもの。
- **精算表（せいさんひょう）**: 決算整理を反映しながら、試算表から B/S と P/L を作るためのワークシート。
- **貸借対照表（B/S）**: 期末時点で会社が持っている資産・負債・純資産の一覧。
- **損益計算書（P/L）**: 1年間の収益と費用、それらの差額として出てくる利益の一覧。

それぞれのステップが何をしているか、順に見ていきましょう。

### ① 取引 → ② 仕訳

ここは普段の業務で動いている部分。お金や物の動きが発生したら、その都度 **仕訳** を切ります。

```
（例）3/15 商品80,000円を仕入、代金は掛け
借方: 仕入   80,000
貸方: 買掛金 80,000
```

仕訳は発生した順に **仕訳帳** に記録されていきます。仕訳帳は「いつ、どんな取引があったか」が時系列で並ぶ帳簿です。

### ③ 元帳：勘定ごとに集計する

仕訳のままだと、「仕入は今いくらある？」と聞かれてもパッとわかりません。仕訳をすべて読み返して足し算する必要があります。これは現実的でないので、**勘定科目ごとに集計しておく帳簿** を作ります。これが **総勘定元帳（元帳）** です。

仕入勘定の元帳ページを覗くと、こんなふうに並びます。

```
仕入（費用）
─────────────────
3/15  買掛金 80,000
3/22  現金   50,000
3/30  買掛金 30,000
─────────────────
合計（借方）       160,000
```

すべての仕訳の **「仕入」が登場したぶんだけ** がここに集計される、という仕組みです。仕訳と元帳は **同じ情報を、別の角度から並べたもの**。仕訳帳が「時間軸」、元帳が「勘定軸」、と覚えると整理しやすいです。

> 元帳の摘要欄で相手勘定が複数あるときは、まとめて **諸口（しょくち）** と書く習慣もあります。詳しくは [振替と相手勘定](/articles/furikae-aitekanjo) を読んでみてください。

### ④ 試算表：残高を一覧化

期末（または月末）になったら、元帳の各勘定の **残高** を一覧表にまとめます。これが **試算表**。

試算表は「全勘定の残高を借方・貸方に並べた表」で、**借方合計と貸方合計が一致する** ことを確認するチェックポイントになります。複式簿記のルール（[複式簿記とはなにか](/articles/double-entry) 参照）どおりに仕訳が切れていれば、試算表でも借方＝貸方になるはず。

ここで合わなかったら、どこかの仕訳に書き間違いがある証拠です。

### ⑤ 精算表：決算整理を反映する

試算表は「仕訳のまま集計した姿」ですが、決算ではこれに **決算整理仕訳** を加えて正しい姿にする必要があります（[決算とは何か](/articles/kessan) 参照）。

そのためのワークシートが **精算表**。8桁精算表なら、

| 勘定科目 | 試算表 借方 | 試算表 貸方 | 修正（整理）借方 | 修正（整理）貸方 | P/L 借方 | P/L 貸方 | B/S 借方 | B/S 貸方 |

という8列の表で、左から右へと **決算整理仕訳の影響をたどりながら** 各勘定が B/S か P/L かに振り分けられていきます。

3級の **第3問** はこの精算表（または財務諸表）作成が頻出です。形式は違っても、やっていることは「試算表 → 決算整理を加えて → B/S と P/L に分ける」だけ。

### ⑥ B/S と P/L：最終成績表

精算表の右端に出てくるのが、最終成果物の **貸借対照表（B/S）** と **損益計算書（P/L）** です。

**貸借対照表（B/S）** は **期末時点のスナップショット**。

```
（B/S の構造）
─────────────────
資産    │ 負債
        │
        │ 純資産
─────────────────
        ↑
資産＝負債＋純資産（会計等式）
```

**損益計算書（P/L）** は **その期間のフィルム**。

```
（P/L の構造）
─────────────────
費用    │ 収益
─────────────────
利益    │
（差額）│
─────────────────
        ↑
収益−費用＝利益
```

P/L で出た利益は、B/S の「繰越利益剰余金（純資産）」に加算されます。これで翌年の出発点になり、次の会計期間の処理が始まります。

### 試験ではどこが出る？

3級の試験では、このパイプラインのうち：

- **第1問（45点）** = 主に **②仕訳** のスキル
- **第2問（20点）** = ③元帳（勘定記入）や補助簿の問題
- **第3問（35点）** = ⑤精算表 / ⑥財務諸表 の作成

「全部のステップを書け」とは出ません。**ある段階のスキル** がピンポイントで問われます。だから「第1問は仕訳特化、第3問は集計特化」と切り分けて学習するのが効率的です。

## D. ポイント整理

- 簿記は **取引→仕訳→元帳→試算表→精算表→B/S・P/L** の一方通行のパイプライン
- **仕訳帳は時間軸、元帳は勘定軸**。同じ情報の並べ方が違うだけ
- **試算表** で借方＝貸方のチェック。合わなかったら仕訳に誤り
- **精算表** で決算整理を反映して B/S と P/L へ振り分け
- 試験は **段階ごとに別の問題** で出る。第1問＝仕訳、第3問＝精算表／財務諸表
