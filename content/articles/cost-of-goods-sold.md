# 売上原価の算定 — しーくり、くりしー

> 「しーくり、くりしー」って呪文みたいですが、これは決算整理の **絶対覚える型** のひとつです。今日はこの2行の意味を、仕入勘定の動きで完全に納得しに行きます。

## A. ストーリー

先輩がカフェ「Bean Garden」を1年間やってきました。期首（4/1）に倉庫には ¥30,000 ぶんの豆が残っていました。1年間で ¥600,000 ぶんの豆を仕入れて、売れ残った豆が期末（3/31）に ¥50,000 だったとします。

さて、この1年で **実際に売れた豆の原価（売上原価）はいくらでしょうか？**

```
期首にあった ¥30,000 + 当期仕入 ¥600,000 − 期末に残った ¥50,000 = ¥580,000
```

この ¥580,000 が **売上原価**。当期に売れた商品分のコストです。簿記3級では、この計算を **仕訳でやらせる** のが第3問の頻出パターン。それが「しーくり、くりしー」です。

## B. 用語の整理

- **三分法（さんぶんぽう）**: 商品の売り買いを「仕入」「売上」「繰越商品」の3勘定で記録するルール。3級ではこれを使います。
- **仕入（しいれ）**: 商品を買ったときに使う **費用** の勘定。期中はずっと借方に積み上がっていきます。
- **繰越商品（くりこししょうひん）**: 期末に残った在庫を翌期に持ち越すための **資産** の勘定。期中は動かず、決算整理だけで動く特殊な勘定。
- **売上原価（うりあげげんか）**: 当期に売れた商品分のコスト。「期首+当期仕入−期末」で計算します。
- **しーくり、くりしー**: 売上原価を仕入勘定で算定するための決算整理2行の暗記フレーズ。詳しくは下で。

## C. 本論

### なぜ決算整理が必要なのか

3級では商品を仕入れたとき、こんな仕訳を切ります。

```
（期中の仕入）
借方: 仕入   80,000
貸方: 買掛金 80,000
```

仕入勘定（費用）にどんどん積み上がっていきます。1年が終わるころには、仕入勘定の残高は **¥600,000**（当期に仕入れた豆の総額）になっています。

ここで問題。**「仕入勘定 = 売上原価」ではない** んです。なぜなら、

- 期首に倉庫に残っていた ¥30,000 ぶんは、**前期に仕入れたけど今期に売れた** かもしれない（→今期の売上原価に入るべき）
- 当期仕入の ¥600,000 のうち、期末に ¥50,000 ぶんが **売れ残っている**（→今期の売上原価から外すべき）

仕入勘定の残高をそのまま売上原価にしてしまうと、この **2つのズレ** が反映されません。これを直すのが決算整理です。

### しーくり、くりしーの2行

正解の仕訳はこの2行。

```
（1行目：し→くり）
借方: 仕入       30,000   ← 期首商品を仕入へ
貸方: 繰越商品   30,000

（2行目：くり→し）
借方: 繰越商品   50,000   ← 期末商品を繰越商品へ
貸方: 仕入       50,000
```

呼び方の由来：

- **「しーくり」** = 「**仕**入」の借方／「**繰**越商品」の貸方（し→くり）
- **「くりしー」** = 「**繰**越商品」の借方／「**仕**入」の貸方（くり→し）

仕入勘定で計算が進むので、**最終的に仕入勘定の残高 ＝ 売上原価**となります。

### 1行目（しーくり）の意味

```
借方: 仕入       30,000
貸方: 繰越商品   30,000
```

この行は **「期首にあった商品 ¥30,000 を仕入勘定に足す」** 操作です。

- **借方の仕入 30,000**: 期首の在庫は当期に売れたかもしれないので、当期の売上原価候補に加える。費用が増えるので借方。
- **貸方の繰越商品 30,000**: 繰越商品（資産）は期首時点で ¥30,000 残っていました。これを取り崩します。資産が減るので貸方。

これで **繰越商品はゼロ**、仕入勘定は ¥600,000 + ¥30,000 = **¥630,000** になります。

### 2行目（くりしー）の意味

```
借方: 繰越商品   50,000
貸方: 仕入       50,000
```

この行は **「期末に残っている商品 ¥50,000 を仕入勘定から差し引いて、来期に持ち越す」** 操作です。

- **借方の繰越商品 50,000**: 期末に売れ残っている ¥50,000 ぶんを資産として翌期に持ち越す。資産が増えるので借方。
- **貸方の仕入 50,000**: 売れ残った商品は当期の売上原価ではないので、仕入勘定から取り除く。費用が減るので貸方。

これで **繰越商品は ¥50,000**（来期の期首在庫として持ち越し）、仕入勘定は ¥630,000 − ¥50,000 = **¥580,000**（売上原価）に確定します。

### 仕入勘定の最終残高 = 売上原価

2行終わった後、仕入勘定の借方残高は **¥580,000**。これが当期の売上原価です。先ほどの算式と一致します。

```
期首 30,000 + 当期仕入 600,000 − 期末 50,000 = 580,000
```

簿記3級では「売上原価は仕入勘定で算定する」と問題文に書かれることが多く、この場合は **しーくり、くりしー** の2行を切れば、自動的に仕入勘定の残高が売上原価になります。

### 「売上原価勘定で算定する」というパターン

たまに「売上原価勘定（pl_summary や売上原価という別勘定）で算定する」と指示される問題もあります（2027シラバスでは出題範囲）。基本の考え方は同じで、仕入勘定の代わりに「売上原価勘定」を使うだけ。出会ったときに問題文の指示に従えば対応できます。

## D. ポイント整理

- 売上原価 = **期首商品 + 当期仕入 − 期末商品**
- 仕訳は **「しーくり、くりしー」** の2行（決まり文句として暗記）
- 1行目（し→くり）：**期首** を仕入へ（借方仕入／貸方繰越商品）
- 2行目（くり→し）：**期末** を繰越商品へ（借方繰越商品／貸方仕入）
- 2行後の **仕入勘定の残高がそのまま売上原価**
- 繰越商品は普段は動かない勘定。決算整理だけで動く特殊枠
